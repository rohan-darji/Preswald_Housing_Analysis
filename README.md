# Housing Analysis

This repository contains code and resources for analyzing housing data using Preswald.

## Live Demo

You can view and interact with the project live at: [https://preswald.app/p18317b88](https://preswald.app/p18317b88)


## Project Structure
- `data/` - Contains datasets used for analysis.
- `images/` - Contains images and visualizations.
- `hello.py` - Main script for analysis.
